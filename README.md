# Netflix-data-analysis
Exploratory Data Analysis (EDA) of Netflix's content catalogue using Python, Pandas and Matplotlib.
Netflix Content Analysis

 Overview

This project explores Netflix's content catalogue using Exploratory Data Analysis (EDA) with Python.

The analysis focuses on identifying trends in content type, release year, country, genre, ratings and directors while demonstrating data cleaning, visualization and business insight generation.


 Objectives

This project answers the following questions:

- What proportion of Netflix's catalogue consists of Movies versus TV Shows?
- Which countries contribute the most content?
- Which genres dominate Netflix?
- How has Netflix's catalogue evolved over time?
- Which ratings are most common?
- Which directors have contributed the most titles?



 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab



 Data Cleaning

The dataset was cleaned before analysis by:

- Removing unnecessary unnamed columns
- Checking for missing values
- Verifying data types
- Preparing the dataset for analysis



 Analysis Performed

- Movies vs TV Shows
- Top Producing Countries
- Most Popular Genres
- Content Released by Year
- Ratings Distribution
- Top Directors



 Key Insights

- Movies significantly outnumber TV Shows.
- The United States contributes the highest amount of Netflix content.
- Drama and International Movies dominate the catalogue.
- Netflix experienced rapid catalogue growth after 2015.
- Mature-rated content (TV-MA) forms the largest share of the platform.
